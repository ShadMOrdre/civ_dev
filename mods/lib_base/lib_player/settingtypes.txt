#lib_player settings


#If enabled, will use MT v5.0+ built in translation system.  Disable will use intllib, mod provided intllib, or none, in that order. Defaults to false.
lib_player_engine_translation (If true, will use the MT v5.0+ built in translation system.  Disable will use intllib, mod provided intllib, or none, in that order. Defaults to false. true/false) bool false

#If enabled, will disable most features of lib_player, to provide a minimal experience for development and simple gameplay.
lib_player_lite (If true, disables all player features except armor, appearance, hunger and the default player_api features. true / false) bool true
